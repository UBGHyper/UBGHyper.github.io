Chrome Extensions

Bypassing

If your school administrator is blocking access to the chrome webstore then there are two options for downloading them.

1. DIY

Begin with downloading your desired chrome extension on a separate non blocked school account. Then turn on developer in the extensions settings and remember the sting of characters below the chrome extensions name. Then navigate to C:\Users\USERHERE\AppData\Local\Google\Chrome\User Data\Profile 1\Extensions (could not be profile 1). Click on the folder that is named the same string of characters as you saw earlier. Copy the folder and paste it somewhere (prob documents). Then enable extension dev mod on the school computer and navigate to the file directory from earlier. Open the file and click on the version file (usually named something like 1.x.x.1_0). Copy the directory of that file and go back to google, click on pack extension and then put the directory of version file from earlier there. Now drag and drop the version file into google and you are done. If you need help email me at Evest356@gmail.com.
